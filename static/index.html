The Flow

From signup to shipped

01
👤

Register

Create an account via POST /auth/register. Password is hashed with PBKDF2-SHA256 and stored in PostgreSQL.

02
🔑

Login & Get Token

Authenticate via POST /auth/login. Receive a signed JWT containing your user ID, email, and role.

03
🛡️

Authenticated Requests

Send Bearer token in every request. The RBAC guard verifies your role before allowing access to protected task endpoints.

04
⚡

Manage Tasks

Create, read, update, delete tasks. Managers see all tasks; members only see their own. Database responds instantly.

Access Control

Two roles. Clear boundaries.

Manager
👑
  • ✓ View all tasks (any user)
  • ✓ Create tasks for any user
  • ✓ Assign & reassign tasks
  • ✓ Update / delete any task
  • ✓ List all registered users
  • ✓ Change user roles
VS
Member
👤
  • ✓ View own tasks only
  • ✓ Create tasks for self
  • ✓ Update own task status
  • ✓ Delete own tasks
  • ✗ Access other users' tasks
  • ✗ Change roles or assign users
